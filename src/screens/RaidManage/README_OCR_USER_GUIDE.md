## 스크린샷 OCR 혈맹원 감지 기능 - 사용자 가이드

### 🎯 기능 개요

게임 스크린샷을 업로드하면 화면에 있는 혈맹원 이름을 자동으로 인식하여 드래프트에 추가하는 기능입니다.

### 📋 사용 방법

#### 1단계: OCR 버튼 클릭

- **위치**: 보스 탭 → "드랍 아이템 목록" 섹션 우측
- **버튼**: `📸 스크린샷 OCR`

#### 2단계: 스크린샷 선택

다음 방식 중 하나로 선택:

**방법 A: 클릭으로 선택**

- 드래그 영역을 클릭
- 파일 선택 창에서 이미지 파일 선택

**방법 B: 드래그 드롭**

- 스크린샷 파일을 드래그해서 영역에 드롭
- 자동으로 미리보기 표시

**지원 파일**:

- 형식: JPG, PNG, GIF, BMP 등
- 최대 크기: 5MB

#### 3단계: 감지 실행

- "✨ 혈맹원 감지" 버튼 클릭
- 백엔드에서 OCR 처리 (수초 소요)
- 감지된 이름 목록 표시

#### 4단계: 결과 확인 및 수정

**자동 매칭된 항목**:

- 녹색 표시 (신뢰도 60% 이상)
- 혈맹원 이름이 자동으로 선택됨

**미분류 항목**:

- 노란색 표시
- 수작업으로 선택 필요

**수정 방법**:

1. "▶ 대안" 버튼으로 대체 선택지 확인
2. 원하는 혈맹원 클릭으로 선택
3. "✕ 미분류" 버튼으로 제외

#### 5단계: 완료

- "✓ 선택 완료" 버튼 클릭
- 선택된 혈맹원이 드래프트 행으로 추가됨
- 각 행의 "루팅자" 필드에 이름이 채워짐

### 💡 팁

**신뢰도가 낮은 경우:**

- 게임 해상도가 낮으면 인식 정확도 감소
- 밝기/명도가 너무 높거나 낮으면 인식 어려움
- 게임 설정에서 해상도를 높이면 개선될 수 있음

**헷갈리는 글자:**
다음 글자들은 OCR에서 혼동될 수 있습니다:

- `O(영어) ↔ 0(숫자)`
- `l(소문자L) ↔ I(영어) ↔ 1(숫자)`
- `S ↔ 5`
- 한글: `이 ↔ 아`, `오 ↔ 영`

이 경우 수동으로 대안 목록에서 선택하세요.

**혈원 리스트에 없는 사람:**

- OCR이 감지했지만 혈맹에 없는 이름
- "✕ 미분류"로 선택하여 제외
- 수동으로 아이템명만 입력

### ⚙️ 주의사항

1. **파일 크기**: 5MB 초과 파일은 업로드 불가
2. **네트워크**: 백엔드 서버 응답 대기 필요
3. **정확도**: OCR 기술의 한계로 100% 정확할 수 없음
4. **수정 필수**: 감지 후 항상 결과 확인 및 수정

### 🔄 OCR 재처리

이미지를 변경하려면:

1. "다시 선택" 버튼 클릭
2. 다른 스크린샷 선택
3. 다시 감지 실행

### ❌ 문제 해결

**"이미지에서 이름을 찾을 수 없습니다" 오류**

- 스크린샷이 게임 창이 아닐 수 있음
- 스크린샷 일부가 잘린 경우
- 해상도가 너무 낮은 경우
- → 다른 스크린샷으로 재시도

**"API 오류" 또는 "처리 중 오류"**

- 백엔드 서버 문제
- 네트워크 연결 끊김
- → 새로고침 후 재시도

**감지된 이름이 정확하지 않음**

- 대안 목록에서 올바른 혈맹원 선택
- 혈맹원이 없으면 "미분류"로 제외

### 📝 워크플로우 예시

```
1. 게임에서 혈맹원 정보 보기 화면 스크린샷
2. "📸 스크린샷 OCR" 클릭
3. 스크린샷 파일 선택
4. "✨ 혈맹원 감지" 클릭
5. 감지된 목록 확인
   ├─ 녹색(자동매칭): 그대로 사용 ✓
   ├─ 노란색(미분류): 대안 선택 또는 제외
   └─ 오류: 수동으로 올바른 이름 선택
6. "✓ 선택 완료" 클릭
7. 드래프트 추가된 행의 아이템명 입력
8. 저장
```

### 🔐 데이터 보안

- 스크린샷은 서버에 임시 저장만 됨
- OCR 처리 후 즉시 삭제
- 이미지 데이터는 로컬에 저장되지 않음
